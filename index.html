PredictRoute

Network Path Prediction

PredictRoute uses a combination of control plane and data plane measurements to build probabilistic models of routing. Using these models, PredictRoute can predict un-measured network paths accurately.

Query PredictRoute's REST API for predicting paths from arbitrary sources and destinations on the Internet. We also provide several helpful tools for the analysis of inter-domain routing. A description of the API and its uses are listed in the documentation.
